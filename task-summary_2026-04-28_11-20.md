### Config Format (JSON)
```json
{

  "name": "DeepSeek V3 Pro",
  "provider": "deepseek",

```

### Features
1. **Interactive Mode** - Default, prompts for missing fields
2. **Non-interactive Mode** - `--no-interactive` flag for CI/CD
3. **Author Detection** - Auto-detect from `OMC_AUTHOR_NAME` env or `git config user.name`
4. **UUID Generation** - 8-char unique ID for each config
5. **Filtering** - Browse by provider, author, or search keywords
