# 竞品对比

> 🔍 客观对比，帮你做选择。

## 对比对象

我们选取了 7 个有代表性的 AI 编程工具，从实际使用角度进行对比。

| 工具 | 类型 | 一句话介绍 |
|------|------|-----------|
| **Oh-My-Coder (OMC)** | 多 Agent 框架 | 国产模型 + 多 Agent 协作，完全开源免费 |
| **Oh-My-OpenCode (OMO)** | CLI 编程工具 | 开源 CLI，支持 75+ 模型 |
| **Claude Code** | AI 编程 CLI | Anthropic 官方 CLI，Agent 能力强 |
| **Gemini CLI** | CLI 编程工具 | Google 官方 CLI，免费开源 |
| **Cline** | VS Code 插件 | IDE 内的 AI 编程助手 |
| **GitHub Copilot** | 编辑器插件 | GitHub 生态的 AI 补全 |
| **Cursor** | AI 原生 IDE | 重新设计的 AI 编程环境 |

## 核心对比

### 价格与付费

| 工具 | 个人免费版 | 付费版 | 是否需要外部 API |
|------|-----------|--------|----------------|
| **OMC** | ✅ 完全免费 | 无付费版 | ✅ 需要（国产模型有免费额度） |
| **OMO** | ✅ 开源免费 | 无付费版 | ✅ 需要 |
| **Gemini CLI** | ✅ 开源免费 | 无付费版 | ✅ 需要 |
| **Claude Code** | ❌ 无免费版 | $20/月 | ⚠️ 需要 Claude Pro |
| **Cline** | ✅ 插件免费 | 无付费版 | ✅ 需要 |
| **GitHub Copilot** | ❌ 无免费版 | $19/月 | ❌ 不需要 |
| **Cursor** | ⚠️ 有限免费 | $20/月 | ❌ 不需要 |

> 💡 OMC 的「免费」是指工具本身免费，API Key 大部分有免费额度（GLM-4.7-Flash 有免费额度）。
> 
> 🎯 **OMC 优势**：免费 + 国产模型 + 中文 + 开源，vs Claude Code ($20/月) / Gemini CLI (仅 Google 模型)

| **OMC** | ✅ | ❌ | ✅ 12 家 | ✅ 原生中文 |
| **OMO** | ⚠️ 部分模型需要 | ⚠️ 部分 | ⚠️ 有限 | ⚠️ |
| **Gemini CLI** | ❌ | ✅ 需要 | ❌ | ⚠️ |
| **Claude Code** | ❌ | ✅ 需要 | ❌ | ⚠️ |
| **Cline** | ⚠️ 配置后可用 | ⚠️ 部分 | ❌ | ⚠️ |
| **Copilot** | ❌ | ✅ 需要 | ❌ | ⚠️ |

> 这是 OMC 最大的差异化优势：**国内用户不需要翻墙，不需要付费订阅，开箱即用。**
> 
> 🎯 **OMC 优势**：vs Claude Code（$20/月 + 需翻墙）/ Gemini CLI（仅 Google 模型 + 需翻墙）

### 多 Agent 能力

| 工具 | Agent 数量 | Agent 协作 | 自定义 Agent | 工作流编排 |
|------|-----------|-----------|-------------|-----------|
| **OMC** | 31 个 | ✅ 多 Agent 协作 | ✅ 代码扩展 | ✅ 7 种内置工作流 |
| **Claude Code** | ~10 个 | ✅ | ❌ | ❌ |
| **OMO** | 无 | ❌ | ❌ | ❌ |
| **Gemini CLI** | 无 | ❌ | ❌ | ❌ |
| **Cline** | 1 个 | ❌ | ❌ | ❌ |
| **Copilot** | 无 | ❌ | ❌ | ❌ |
| **Cursor** | 无 | ❌ | ❌ | ❌ |

> 多 Agent 协作是 OMC 的核心架构。简单任务用单 Agent 就够，复杂任务（如「从零搭建一个 API」）需要多个 Agent 分工：探索→分析→设计→实现→验证。这种分工模式更接近真实团队协作。
> 
> 🎯 **OMC 优势**：31 个专业 Agent vs Claude Code（~10 个）/ Gemini CLI（无）

## 结论

### OMC 的目标用户

1. **国内开发者**：不想翻墙、不想付美元、想用国产模型
2. **学生 / 独立开发者**：预算有限，希望零成本使用 AI 编程
3. **多 Agent 爱好者**：需要多个 Agent 协作完成复杂任务
4. **注重隐私的开发者**：希望代码不出本机

### OMC 的核心差异化

**国产模型 + 零成本 + 多 Agent 协作 + 完全开源 + 中文优化**

这个组合目前没有其他工具能做到：
- **Claude Code**：$20/月 + 需翻墙 + 仅 Claude 模型 + ~10 个 Agent + 闭源
- **Gemini CLI**：免费开源 + 需翻墙 + 仅 Google 模型 + 无 Agent 能力
- Copilot / Cursor：需要翻墙 + 付费，无多 Agent
- Cline：IDE 集成好，但国产模型支持弱
- OMO：开源 CLI，但无多 Agent 协作
- AutoGen：企业级框架，但无国产模型优化，上手门槛高

**OMC 的独特价值**：
- ✅ **vs Claude Code**：免费 + 国产模型 + 31 个 Agent + 开源
- ✅ **vs Gemini CLI**：12 家国产模型 + 31 个 Agent + 中文优化
- ✅ **vs 其他**：国内直连 + 零成本 + 完全可控

### 什么时候不选 OMC

诚实地说，以下场景 OMC 不是最佳选择：

- 你只用 IDE 插件，不习惯命令行 → **选 Cline 或 Copilot**
- 你追求最强代码补全体验 → **选 Cursor 或 Copilot**
- 你的团队已经在用 GitHub 生态 → **选 Copilot**
- 你需要最成熟的社区和文档 → **选 Copilot 或 Cursor**

没有最好的工具，只有最合适的工具。OMC 填补的是「国产模型 + 多 Agent + 零成本」这个空白，如果你恰好需要这个组合，欢迎试试。
