### New Command: `omc local chat`

```bash
# Interactive chat (default qwen2:7b)
omc local chat

# Specify model
omc local chat llama3:8b

# With system prompt
omc local chat --system "你是Python专家"

# Adjust temperature
omc local chat --temp 0.9

# Non-streaming mode
omc local chat --no-stream
```

### Chat Features
- **Auto Service Detection**: Checks if Ollama is running at http://localhost:11434
- **Model Verification**: Validates requested model is installed
- **Streaming Output**: Real-time token streaming (default)
- **Interactive Commands**:
  - `/exit`, `/quit`, `/q` - Exit chat
  - `/clear` - Clear conversation history
  - `/help` - Show commands
